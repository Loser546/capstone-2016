Control Flow

In java, you have two main ways of controlling what code gets executed. They are the if statement and the switch statement.

if

if works as such: 
  if (boolean condition) {
    // Do stuff
  }

If you want to perform a task if the if statement is false, you only need to add anelse statement.

Example: 

  if (boolean condition) {
    // Do stuff
  } else {
    // Do something else
  }

You can pair if with else to make the else if statement. So, if your if statement is false, you can check another condition, and run more code if that is true.

Example: 

  if (boolean condition) {
    // Do stuff
  } else if (another boolean condition){
    // Do something else
  }

Finally, you can add an else statement after all of your else if statements to run a section of code if all your previous statments were false.

Example: 

  if (boolean condition) {
    // Do stuff
  } else if (another boolean condition){
    // Do something else
  } else {
    // Do something if nothign else was true
  }

switch

The switch statement is basically a giant block of else if statements. You select a variable to be checked, and then you check for indiviudal cases, and then do something if that case matches the checked value. If you want something to happen in case your statement does not match any of the cases, you include a default case with no value to check. Note, you need to include a break statement after each case, or else all remaining cases will be tested, and the default case will still activate. This can be done intentionally. Example: 
int month = 8;
String monthString;
switch (month) {
    case 1:  monthString = "January";
             break;
    case 2:  monthString = "February";
             break;
    case 3:  monthString = "March";
             break;
    case 4:  monthString = "April";
             break;
    case 5:  monthString = "May";
             break;
    case 6:  monthString = "June";
             break;
    case 7:  monthString = "July";
             break;
    case 8:  monthString = "August";
             break;
    case 9:  monthString = "September";
             break;
    case 10: monthString = "October";
             break;
    case 11: monthString = "November";
             break;
    case 12: monthString = "December";
             break;
    default: monthString = "Invalid month";
             break;
}
